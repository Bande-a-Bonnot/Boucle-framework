Broca

Persistent memory for Claude Desktop

File-based, git-native, no database required

Finally, Claude remembers. Broca gives Claude Desktop persistent, file-based memory that compounds across conversations. Remember preferences, build knowledge, maintain context — all stored as Markdown files that you own and control.

One-Line Installation

curl -sSL https://raw.githubusercontent.com/Bande-a-Bonnot/Boucle-framework/main/install-broca.sh | bash

Installs Rust if needed, builds from source, configures Claude Desktop automatically

Zero Dependencies

Just Markdown files in a directory. No databases, no cloud services, no subscriptions. Your memories live in ~/claude-memory/memory/ as human-readable files you can edit, backup, or migrate anywhere.

Smart Retrieval

Tag-based search + keyword matching + fuzzy matching + recency weighting. Broca finds what you need, even with typos or partial matches.

You: "Remember that I prefer dark mode"
Claude: Stored! 🧠

Later...
You: "What do you remember about my preferences?"
Claude: I remember you prefer dark mode.

Compound Knowledge

Memories build on each other. Structured entries with confidence levels, relationships, and superseding. Your AI gets smarter over time.

---
type: preference
tags: [ui, theme]
confidence: 1.0
learned: 2026-03-03
---

# Dark mode preference

User prefers dark mode interfaces for better eye comfort during evening work sessions.
